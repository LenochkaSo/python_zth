# Set
Unordered collection of unique, immutable objects.
Set literals
```python
# empty set
empty_set = set() . # not {} since it resevered for dict...

# set of objects
int_set = {1, 4, 2, 7}

# set of hetrogenous objects
hetro_set = {1, False, 4.3}
```
