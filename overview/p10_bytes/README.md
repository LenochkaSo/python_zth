
# bytes
Immutable sequence of bytes
- immutable - means you can't modify str content
- sequence - follows the sequence protocol
## bytes literals
```python
b'this is a bytes literal'
b"this is a bytes literal too"

#with hex values (whitespace ignored)
b"0xCA 0xFE"
```
